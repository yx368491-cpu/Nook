# ADR-018 · 自托管 Inter + JetBrains Mono WOFF2（绝不 Google CDN）

## Status

✅ **Accepted**

## Date

2026-06-27

## Context

`../../01_Product/Nook-SPEC.md` F-UI-05 和 `../Nook-ARCHITECTURE.md` § 7 要求字体自托管。原因：国内 GFW 不稳定 Google Fonts CDN；强行引用会导致部分用户看不到正确字体，且违反产品「中国大陆可访问」约束（§ 9.9）。

## Decision

- `public/fonts/` 放置 Inter + JetBrains Mono WOFF2 自托管。
- CSS `@font-face` 指向本地路径，不引用任何外部 CDN。
- 离线时字体显示正常（因为自托管在 CF Pages 上）。
- **永远不允许**引用 Google Fonts CDN。

## Alternatives Considered

| 方案 | 优点 | 缺点 |
|------|------|------|
| **Google Fonts CDN** | 零维护；自动更新 | 国内不稳定；违反产品约束 |
| **自托管 WOFF2（选中）** | 可靠；离线可用 | 需要手动下载字体文件；更新需手动 |
| **系统字体回退** | 零依赖 | 视觉一致性无法保证 |

## Consequences

- M1-7 需要把 2 个 WOFF2 文件放入 `public/fonts/`。
- CSS 中 `@font-face` 指向本地路径。
- Lighthouse 检测不会出现 Google Fonts 请求（AC.AC.fonts 验证项）。

## 是否允许未来修改

**中可逆性**——可迁移到其他 CDN（如 unpkg），但永远不允许 Google Fonts。
